Hi there. Mode represents a webpack base for a modern front-end web application.

Mode transpiles the latest JavaScript with Babel and loads styles from Sass/PostCSS. Other than that, it's minimalist and agnostic, adding no frameworks or libraries. Production and development modes are enabled, and environment variables can be set dynamically. Mode copies static assets such as images and documents with a dedicated /static directory - no need to import them in your JavaScript.

Check your browser's console to read a welcome message, check mode, and inspect a deliberate bug to test source mapping.

Inspiration: Robin Wieruch's node.js tutorials [1]

How to use

To recreate this site, use this branch of the project.
npm start to run in dev mode.
npm run build for a production build into a /build directory, with a main.css stylesheet.
npm run build:int-css for a production build with css injected into the head

Webkit analytics tools are included:
npm run build:analyze generates a report and a statistics visualisation of the build.

Hi there. This application represents a basic set up for a modern front-end webapp, based on rwierucl's ideas. Developed using node js, it uses webpack to bundle js assets and transpile with babel.

Both production and development modes are enabled, webpack code optimised with a common configiration. Environment variables can be set dynamically, and source mapping is enabled to aid debugging.

Check your browser's console to see if this is in development or production mode, read a welcome message, and see a deliberate bug to test source mapping.

How to use

To recreate this site, use this branch of the project.
npm start to run in dev mode.
npm run build for a production build into a build/ folder.

Webkit analytics tools are included:
npm run build:analyze generates a report and a statistics visualisation of the build.

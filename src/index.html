Hi there. This application's development and production uses node js to run webpack, which with babel enables a basic set up for a front-end app, according to rwierucl.

Both production and development modes are enabled, with the code optimised with a common configiration. Environment variables can be set dynamically.

How to use

To recreate this site, use this branch of the project.
npm start to run in dev mode.
npm run build for a production build into a build/ folder.

Webkit analytics tools are included:
npm run build:analyze generates a report and a statistics visualisation of the build.

Hi there. Mode represents a base for a front-end web application built in webpack.

Mode transpiles modern JavaScript with Babel and loads styles from Sass/PostCSS. It's opinionated yet minimalist, adding no frameworks or libraries. Mode copies static assets such as images and documents with a dedicated /static directory - no need to import them in your JS.

Production and development modes are enabled. Check your browser's console for more information.

Inspiration: Robin Wieruch's 'modern JS' tutorials [1]

How to use

To recreate this site, use this branch of the project.
For a more minimalist front-end, use this branch.

npm start to run in dev mode.
npm run build for a production build into a /build directory, with a main.css stylesheet.
npm run build:int-css for a production build with css injected into the head

Webkit analytics tools are included:
npm run build:analyze generates a report and a statistics visualisation of the build.

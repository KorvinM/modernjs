Hi there. This foundation for a modern front-end webapp, built in webpack, transpiles the latest JavaScript with babel and compiles styles from SASS.

Other than that, it's agnostic, adding no frameworks or libraries. Production and development modes are enabled, and environment variables can be set dynamically.

Check your browser's console to read a welcome message, check mode, and inspect a deliberate bug to test source mapping.

Inspired by Robin Wieruch's node.js tutorials [1]

How to use

To recreate this site, use this branch of the project.
npm start to run in dev mode.
npm run build for a production build into a build/ folder, with a main.css stylesheet.
npm run build:int-css for a production build with css injected into the head

Webkit analytics tools are included:
npm run build:analyze generates a report and a statistics visualisation of the build.
